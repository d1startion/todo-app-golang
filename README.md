# Todo App (Golang)

REST API для управления задачами с авторизацией и слоистой архитектурой.

---

## 🚀 Возможности

* Регистрация и аутентификация пользователей (JWT)
* CRUD операции для задач
* Валидация входных данных
* Централизованная обработка ошибок
* Unit-тесты (testing, testify)
* CI/CD (GitHub Actions)
* Docker-контейнеризация

---

## 🧱 Архитектура

Проект построен по принципу разделения ответственности:

* **handler** — обрабатывает HTTP-запросы (парсинг, ответы)
* **service** — содержит бизнес-логику приложения
* **repository** — отвечает за работу с базой данных

Такой подход:

* упрощает тестирование
* улучшает читаемость кода
* облегчает масштабирование проекта

## ▶️ Запуск проекта

```bash
docker-compose up --build
```

### Локально:

```bash
go run main.go
```

## 📡 Основные endpoints

### Аутентификация

* `POST /register`
* `POST /login`

### Задачи

* `GET /tasks`
* `POST /tasks`
* `PUT /tasks/{id}`
* `DELETE /tasks/{id}`
